SemiProject - ccookat

### 📖프로젝트 소개
모바일 버전으로만 운영되고 있는 쿠캣 사이트를 웹 버전으로 개발해보고자 했습니다. 회원가입 및 로그인, 상품 카테고리 게시판과 상품페이지, 장바구니와 구매, 리뷰와 질문답변 등 쇼핑몰의 기본 페이지를 구현했습니다.

- 개발기간 : 2022.08.02 ~ 2022.08.10
- 인원/역할 : 5명/팀장
